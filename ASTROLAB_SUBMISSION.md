# 🚀 Astrolab Submission: Opportunity Aggregator

### Pitch
The Opportunity Aggregator is an AI-powered personal radar that scans the tech landscape for hackathons, grants, and events. It doesn't just list them; it ranks them against your personal professional profile using Gemini 3.1.

### Key Technical Achievements
1.  **Commute-Built:** The entire project was scaffolded and refined during bus commutes, proving that meaningful engineering can happen anywhere.
2.  **Smart Scraper:** Modular architecture supporting MLH, Devpost, Hack Club, TabNews, and GitHub Jobs.
3.  **Proactive Alerts:** A background radar that filters for "Elite" matches (>90%) and pushes them to Discord.
4.  **BYOK Architecture:** Scales infinitely by allowing users to use their own API keys.
5.  **Multi-Tier AI:** Automatic fallback from Gemini to OpenRouter models ensures the radar never goes dark.
